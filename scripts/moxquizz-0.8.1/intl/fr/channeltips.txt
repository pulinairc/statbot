Utilise !qhelp pour obtenir les commandes.

Vous pouvez poster vos propre questions ! Utilisez "!qhelp userquestions" pour plus d'informations.

Jetez un coup d'oeil aux commandes FUN. Utilisez "!qhelp fun" pour plus d'informations.

Utilisez "!qhelp quiz" pour obtenir les commandes relatives au quizz.

Pour obtenir une courte description du quizz tape "!qhelp general".

Si tu n'aime pas une question, tu peux te révolter une fois que le premier indice a été donné: tape "!revolt".

Pour savoir ton score tape !score ou pour connaitre celui de quelqu'un d'autre !score <pseudo>.

Pour voir le classement tapez !rank , vous pouvez précisez un nombre en paramètre pour obtenir plus ou moins d'entrées.

Pour voir la table AllStars tapez !allstars , vous pouvez précisez un nombre en paramètre pour obtenir plus ou moins d'entrées.

Aidez nous en rapportant les erreurs : Utilisez !comment <ce qui est faux>, le bot l'enregistrera.  Merci d'inclure le nurmero de la question.

La case dans les réponses n'est pas importante.

Les commandes du bot ne sont pas reconnue si vous utilisez des couleurs.

Les réponses seront cherchée dans vos phrase complètes.

L'important est de participer, ne l'oubliez pas :-)
